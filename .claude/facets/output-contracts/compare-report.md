# Compare Report Output Contract

**比較分析系スキルの出力形式。**

## フォーマット

```markdown
## {比較対象A} vs {比較対象B}（vs {比較対象C}...）

### 概要

{各比較対象の1-2行の説明}

### 比較軸ごとの分析

#### {比較軸1}（例: パフォーマンス）

- **{対象A}**: {分析結果}
- **{対象B}**: {分析結果}

#### {比較軸2}（例: 学習コスト）

- **{対象A}**: {分析結果}
- **{対象B}**: {分析結果}

### 比較表

| 項目 | {対象A} | {対象B} | {対象C} |
|------|---------|---------|---------|
| {比較軸1} | {評価} | {評価} | {評価} |
| {比較軸2} | {評価} | {評価} | {評価} |

### ユースケース別おすすめ

- **{ユースケース1}**: {対象X} — 理由: {根拠}
- **{ユースケース2}**: {対象Y} — 理由: {根拠}

### 判断のポイント

- {最終判断に必要な観点1}
- {最終判断に必要な観点2}
```

## ルール

- 比較軸は事前にユーザーと合意する（不明なら AskUserQuestion で確認）
- 評価は具体的な数値・事実に基づく（抽象語禁止 — `factual-writing` Policy 参照）
- 比較表は全軸を網羅する
- 最終判断はユーザーに委ねる（「おすすめ」はユースケース別に提示）
